# Sales Analysis Dashboard (Power BI)
# Project Overview
This is a Sales Analysis Dashboard where I analyzed sales, profit, and order trends. I used different visuals like line charts, bar charts, and maps to identify patterns and business insights.

# Objectives
- Analyze overall sales and profit performance
- Track monthly sales and profit trends (Year-over-Year)
- Identify top-performing categories and subcategories
- Understand customer segments and payment behavior
- Analyze shipping modes and delivery performance

# Key Metrics
- Total Sales: 1.57M
- Total Orders: 22K
- Total Profit: 175K
- Average Shipping Days: 4

# Key Insights
- Consumer segment contributes the highest sales (48%)
- COD is the most used payment mode (43%)
- Office Supplies category generates the highest sales
- Phones, Chairs, and Binders are top subcategories
- West and East regions have the highest sales contribution
- Sales and profit show growth trends over months

# Tools & Technologies Used
- Power BI
- Power Quary
- Excel

# Dashboard Features
- Region filter (Central, East, South, West)
- Monthly Sales & Profit trend (YoY comparison)
- Sales by Category and Subcategory
- Sales by Ship Mode
- Sales Distribution by Segment and Payment Mode
- Geographic Sales Analysis (Map visualization)

# Conclusion
This dashboard helps businesses understand sales performance, identify profitable areas, and make data-driven decisions to improve overall business growth.
